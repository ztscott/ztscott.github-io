Java 如何使用 使用 JDBC 连接 MySQL 数据库，MySQL 8.0 以上版本的数据库连接有所不同：

1、MySQL 8.0 以上版本驱动包版本 mysql-connector-java-8.0.16.jar。

2、com.mysql.jdbc.Driver 更换为 com.mysql.cj.jdbc.Driver。

MySQL 8.0 以上版本不需要建立 SSL 连接的，需要显示关闭。

最后还需要设置 CST。
